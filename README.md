# Socksclassic ERC721 token

Socksclassic is a ERC721 compatible non fungible token.

## Requirements
* python3

## Installation

1. Clone this repo

2. Setup environment
```
python3 -m venv env
source env/bin/activate
```

3. Install dependencies
```
pip install -r requirements.txt
```

## Testing
```
pytest -v test/
```
